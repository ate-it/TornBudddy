# MyApp

A modern Rails application using:

- 🚀 Rails (latest)
- 🐘 PostgreSQL
- 🗺️ Importmaps
- 🎨 Bootstrap 5 (via CDN)
- 🔥 Toast notifications (Bootstrap)
- ✅ RSpec for testing
- 💌 SMTP mailing (via `.env`)
- 📝 Simple Form
- 🛠️ Debugging tools (`better_errors`, `pry`)
- ✅ RuboCop linting
- 🌐 UTC timezone
- 🏁 Seed data included

---

## 🔧 Getting Started

### 1. Clone & Install Dependencies

```bash
git clone https://github.com/your-username/my_app.git
cd my_app
bundle install
```
